# Portfolio Website

Personal portfolio website showcasing my projects, experiments, and ideas.

Built with a focus on clean UI, modular components, and smooth user interactions.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS
- Framer Motion

## Features

- Scroll-snap section layout
- Interactive project cards
- Smooth animations and transitions
- Fully responsive design

## Local Development

```bash
npm install
npm run dev
